LogBrew supports six providers today: Sentry, Vercel, Railway, PostHog, Linear, and Stripe. We are adding new integrations on a rolling basis, and the next one we build is often the one most users ask for.

## Request a new integration

If the developer tool you depend on is not yet supported, send us a short note. Tell us:

- **Which provider** you would like to see integrated
- **What events matter most** to you (errors, deployments, billing, alerts, logs, etc.)
- **How you currently get notified** about those events today

We read every request and use them to prioritize our integration roadmap.

  [support@logbrew.co](mailto:support@logbrew.co)
</Card>

## What's coming

New integrations are released as we finish them. The current shortlist includes the most-requested developer tools across observability, deployment, billing, and project management. Updates land in the App Store and the integration appears in the **Add Project** sheet automatically. No app reinstall needed.

<Tip>
  Already a LogBrew user? Open the app, tap **Settings**, and check **What's New** after each update for the latest integrations and features.
</Tip>
